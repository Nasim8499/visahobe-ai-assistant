# VisaHOBe AI Assistant Feature Roadmap

## Phase 1: Core Dashboard
- Executive command center
- Case pipeline overview
- Country workflow cards
- Client intake status
- Document readiness tracker

## Phase 2: Passport Intake
- Passport image upload UI
- Manual correction form
- OCR provider adapter layer
- MRZ parsing placeholder
- Client profile auto-fill

## Phase 3: Document Generator
- A4 PDF preview canvas
- Visa cover letter template
- Employment/support letter template
- Travel itinerary template
- Client checklist template
- Export review workflow

## Phase 4: AI Assistant
- Perplexity-style chat panel
- Prompt library for visa consultants
- Country-specific workflow guidance
- Client case summarizer
- Employee training assistant

## Phase 5: Operations Hub
- Employee onboarding steps
- Document QA checklist
- Client communication templates
- Daily country update dashboard

## Compliance Guardrails
The system must only support accurate, lawful, client-approved documentation. It must never generate fake identities, fake government records, false bank statements, forged evidence, or misleading application materials.
